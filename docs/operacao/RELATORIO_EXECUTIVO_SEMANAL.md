# Relatório Executivo Semanal

Este arquivo documenta o artefato automático da Operação Tração. A versão atualizada é gerada pelo Dashboard Executivo no link **Baixar relatório semanal**, com o nome `RELATORIO_EXECUTIVO_SEMANAL.md`.

O relatório contém:

- resumo da semana;
- indicadores;
- problemas encontrados;
- oportunidades;
- sugestões;
- missão da próxima semana.

Os valores são calculados no momento do download e usam exatamente a mesma fonte do dashboard. O arquivo não é sobrescrito automaticamente no repositório ou no servidor.
