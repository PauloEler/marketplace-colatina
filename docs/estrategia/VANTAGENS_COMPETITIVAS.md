# Gestão das Vantagens Competitivas

As vantagens estratégicas oficiais estão em [`../../00_CORE/VANTAGENS_COMPETITIVAS.md`](../../00_CORE/VANTAGENS_COMPETITIVAS.md).

## Como validar

| Vantagem | Evidência mínima |
|---|---|
| identidade local | uso recorrente e retorno de moradores de Colatina |
| base operacional | jornadas existentes funcionando com estabilidade |
| confiança | baixa incidência de falhas críticas e compreensão dos estados |
| integração local | tarefas concluídas entre diferentes jornadas |
| relação com empresas | participação e renovação sem perda de transparência |
| evolução disciplinada | Sprints rastreáveis, medidas e com aprendizado registrado |

## Revisão

Cada vantagem deve ser revista trimestralmente. Sem evidência, permanece hipótese e não deve ser apresentada como fato em comunicação pública.
